# Python-Fundamentals-BME
This repository documents my foundational Python learning journey through small biomedical-oriented computational tasks and mini simulations.

The primary goal of this repository is to strengthen:
- computational thinking
- Python programming fundamentals
- debugging and problem decomposition
- biomedical data handling logic
- structured software development practices

The projects intentionally emphasize core programming concepts including:
- variables and data types
- conditional logic
- loops and iteration
- functions
- lists and dictionaries
- modular code organization
- basic signal/data analysis

These implementations represent my early-stage hands-on exploration into programming for biomedical and AI-oriented applications.

## Mini Projects Included

| Project | Concepts Practiced |
|---|---|
| BMI Health Analyzer | Functions, conditions, arithmetic operations |
| Heart Rate Safety Monitor | Threshold logic, condition checking |
| ECG Abnormality Detector | Lists, loops, anomaly detection |
| Step Activity Analyzer | Data aggregation, statistics, iteration |
| Fall Detection Simulator | Sensor threshold simulation, indexing |
